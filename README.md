# CS50 2024 Projects

CS50 é um curso de ciência da computação feito pela Universidade de Havard, que busca ensinar uma base teórica e prática de programação, focada em resoluções de problemas.

O curso é organizado em 10 semanas, cada uma com um conteúdo diferente. O curso começa com programação em blocos e termina com a contrução de um sistema web com o desenvolvimento do frontend, backend, conexão com banco de dados e consumo de APis.

Toda semana possui exercícios em forma de projetos para consolidar o conhecimento. Deixo nesse repositório as minhas resoluções.
